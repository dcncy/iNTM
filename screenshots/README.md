## 📝 截图规范

### 分辨率
- 最小：1200x800
- 推荐：1600x1200 或 1920x1280
- 使用 2x Retina 分辨率以获得更清晰的效果

### 格式
- 使用 PNG 格式
- 压缩图片以减小文件大小（推荐使用 TinyPNG）
- 保持图片质量清晰

### 内容要求
- 使用真实但非敏感的数据
- 隐藏或模糊个人信息（IP 地址可以替换为示例 IP）
- 确保界面干净整洁
- 避免显示错误或异常状态
- 英文版使用英文界面语言
- 中文版使用简体中文界面语言

### 拍摄技巧
1. **窗口截图**
   - 使用 `Cmd + Shift + 4`，然后按空格键
   - 点击要截取的窗口
   - 这样会自动包含窗口阴影

2. **区域截图**
   - 使用 `Cmd + Shift + 4`
   - 拖动选择要截取的区域

3. **使用截图工具**
   - CleanShot X
   - Xnapper
   - ScreenFlow
